# 🚗 Chapacar Management System

Full-stack management system developed for an auto repair workshop.

## 📌 Description

This system was built as a final project for the Programming Analyst degree.

It provides a complete solution for workshop operations, including:

- Stock management
- Workshop task management
- Customer assistance using AI

## 🛠️ Technologies Used

### Frontend
- React
- JavaScript
- CSS

### Backend
- Node.js
- Express

### Database
- MongoDB

## ⚙️ Features

- Manage inventory and stock
- Track workshop jobs and tasks
- Handle customer interactions
- AI-powered assistance for users
